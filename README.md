<h2 align="center">About the Project</h2>

<p align="center">
  StayGames is a <strong>fictional gaming hub</strong> built as a personal project to showcase modern frontend development with React and Vite. 
  It consumes the <a href="https://www.freetogame.com/api-doc">FreeToPlay Games Database API</a> to display hundreds of free-to-play titles 
  across PC and browser platforms.
</p>

<p align="center">
  The focus was on creating a clean, fast, and visually appealing user experience with smooth animations, 
  responsive design, and interactive features.
</p>
